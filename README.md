# Multiplication App

Console app for view the multiplication tables made with NodeJs

```
Options:
      --help     Show help                                             [boolean]
      --version  Show version number                                   [boolean]
  -b, --base     Es la base de la tabla de multiplicar       [number] [required]
  -l, --listar   Define si muestra ó no la tabla de multiplicar
                                                      [boolean] [default: false]
  -h, --hasta    Define el numero limite de la multiplicación
                                                          [number] [default: 10]
```
